jQuery Sortable


A documentation about jQuery UI Framework on how to use Sortable inside a table or list


Include https://code.jquery.com/ui/1.12.1/jquery-ui.js into your header, afterwards, initialize the parent container with the desired id or class name of yours, for example: id="sortable", and include ui-state-default to the elements you wish to be sorted


Include $("desired ID or Class").sortable(); inside your Javascript file, to initialize the Sortable Effect
